# Private Lobby v3 — ND Statistics PWA

## Changes in v3
- Recents / Continue Reading system (persists with localStorage)
- Smart time-based greeting on open
- Formula Hub redesigned: Course cards → drill-in per course
- Copy button on every formula
- YouTube buttons now pill-shaped
- True dark mode (no blue tone)
- Font switching fixed globally
- PWA install banner (Android) + iOS hint in Settings
- App icon: black background + cyan "P"

## Files
- index.html — Complete app
- manifest.json — PWA manifest
- sw.js — Service worker (offline)
- icon-192.svg / icon-512.svg — App icons

## Install
### Android (Chrome): Menu → Add to Home Screen → Install
### iOS (Safari): Share → Add to Home Screen

## Deploy
Host all files on Netlify, Vercel, or GitHub Pages.
